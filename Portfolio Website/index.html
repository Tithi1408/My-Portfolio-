Projects

Most recent work

Hotel Management System

Hotel Management System, is a Flask-based web application designed to streamline hotel operations including guest reservations, room tracking, billing, and analytics. The system features a responsive UI with real-time updates, secure login, and dynamic dashboards powered by SQLite and custom SQL queries to visualize occupancy and revenue insights.

GitHub Repository

Breast Cancer Prediction

This project predicts the presence of breast cancer using a Logistic Regression model for binary classification, identifying tumors as Benign or Malignant based on medical attributes. It uses the Breast Cancer Wisconsin Dataset, which includes features such as radius, texture, and smoothness derived from FNA (Fine Needle Aspiration) images. Built in Python with NumPy, Pandas, and Scikit-learn, the model supports early diagnosis through accurate predictions.

GitHub Repository

NLP Projects

This project explores core concepts in Machine Learning and Natural Language Processing through hands-on implementations. It includes a Naive Bayes Classifier (from scratch), Logistic Regression (with scikit-learn), sentence probability calculations, and a simple language model for word prediction. Built in Python with libraries such as NumPy, Pandas, scikit-learn, NLTK, and spaCy, the repository demonstrates both fundamental algorithms and their applications in text classification and language modeling.

GitHub Repository

Analyzing EV Adoption Trends in Washington State

This project examines electric vehicle (EV) adoption patterns using spatial, temporal, and statistical analysis. It identifies geographic hotspots, forecasts registration trends with ARIMA, and applies clustering to evaluate charging infrastructure gaps. Findings highlight urban dominance in EV adoption, BEVs outperforming PHEVs, and socioeconomic factors shaping adoption patterns.

GitHub Repository
